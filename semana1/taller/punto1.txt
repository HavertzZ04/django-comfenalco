A) Explica con tus palabras para qué se usan los siguientes términos en Python:

        print:
        Se utiliza para mostrar información en la pantalla o consola. Permite que el programa comunique resultados, mensajes de error o cualquier otro tipo de información al usuario.

        if:
        Se usa para tomar decisiones en el código. Permite ejecutar ciertos bloques de código solo si se cumple una condición específica.

        for:
        Se utiliza para repetir un bloque de código un número determinado de veces, iterando sobre una secuencia (como una lista, una cadena o un rango).

        while:
        Se usa para repetir un bloque de código mientras una condición sea verdadera. A diferencia de for, no se sabe de antemano cuántas veces se ejecutará el bucle.

B) Explica con tus palabras para qué se usan los siguientes comandos en git:

        add:
        Se utiliza para añadir cambios en los archivos al área de preparación (staging area). Esto prepara los archivos para ser confirmados (commit) en el repositorio.

        commit:
        Se utiliza para confirmar los cambios en el repositorio. Un commit guarda un conjunto de cambios en el historial del proyecto, creando un punto de control al que se puede regresar si es necesario.

        pull:
        Se usa para actualizar el repositorio local con los cambios del repositorio remoto. Esto asegura que tu copia local esté al día con el trabajo de otros colaboradores.

        push:
        Se utiliza para enviar tus commits locales al repositorio remoto. Esto actualiza el repositorio remoto con tus cambios, haciéndolos accesibles para otros colaboradores.

¿Por qué crees que es importante usar git en un proyecto de programación?

        Es importante usar git porque facilita la colaboración entre múltiples desarrolladores, permite llevar un control de versiones eficiente, ayuda a gestionar y revertir cambios en el código, y asegura que todo el equipo pueda trabajar de manera organizada y sincronizada en un proyecto.
